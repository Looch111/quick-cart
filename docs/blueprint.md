# **App Name**: CoinFlow

## Core Features:

- Balance Display: Display current balance in selected cryptocurrencies and fiat currency equivalents.
- Deposit Functionality: Enable users to deposit cryptocurrency into their CoinFlow wallet.
- Withdrawal Functionality: Enable users to withdraw cryptocurrency from their CoinFlow wallet to an external address.
- Multi-Asset Support: Support for multiple tokens and NFTs within the wallet.
- Transaction History: Provide detailed transaction history for all deposits and withdrawals.
- Secure Authentication: Secure user accounts with password protection. Add local authentication, but without cloud database component

## Style Guidelines:

- Primary color: Vibrant yellow (#FFC72C), reminiscent of gold, signifies value and financial transactions.
- Background color: Dark gray (#222222), provides a clean and modern backdrop that makes the primary color stand out.
- Accent color: A light-yellow (#FFF2CC), serving as a contrasting highlight for interactive elements and important information.
- Headline font: 'Poppins' (sans-serif), offering a modern and geometric feel that pairs well with the app's UI; body font: 'PT Sans' (sans-serif), known for its readability and neutrality
- Custom-designed icons in a minimalist style to represent different cryptocurrencies, wallet functions, and settings.
- Clean and intuitive layout with a focus on ease of use, similar to the provided image. Use of white space to avoid clutter.
- Subtle animations to confirm transactions and other interactions. Smooth transitions between screens for a seamless user experience.
